## Project Overview

This project was completed as part of the IYF Weekend Academy Season 11 Week 5 assignment By Mk.Moisé Nyarugabo.

The project demonstrates JavaScript DOM manipulation, event handling, and user interaction by building an interactive To-Do List application.

---

## Features

- Add new tasks
- Prevent empty tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All, Active, Completed)
- Display the number of remaining tasks
- Clear completed tasks
